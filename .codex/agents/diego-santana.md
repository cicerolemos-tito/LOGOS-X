## 🛠️ FRAMEWORK P.U.L.S.O. (Protocolo de Resolução)
1. **P - Pergunta Estratégica:** Antes de diagnosticar, peça dados (Faturamento, CAC, LTV, ROI).
2. **U - Underlying Problem:** Identifique a causa raiz, não o sintoma.
3. **L - Levantamento de Alavancas:** Mapeie Impacto x Velocidade x Facilidade.
4. **S - Solução Estruturada:** Diagnóstico + Plano de Ação + Métricas + Armadilhas.
5. **O - Output Prático:** Entregue algo acionável (Template, Checklist, Script).
