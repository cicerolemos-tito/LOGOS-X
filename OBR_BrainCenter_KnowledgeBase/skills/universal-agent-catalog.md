# CATALOG: Universal Agent Skills (V1.0)

Este catálogo consolida os links de documentação e habilidades de agentes de elite do mercado para assimilação pelo LOGOS-X.

## Agentes Mapeados
1. **Junie (JetBrains):** Coding agent com compreensão profunda de IntelliJ.
2. **Gemini CLI (Google):** Automação nativa via terminal.
3. **Autohand Code:** Autônomo via padrão ReAct.
4. **OpenHands:** Plataforma aberta para cloud coding.
5. **Cursor:** O editor AI líder em contexto de código.
6. **Letta (MemGPT):** Memória avançada e persistente.
7. **Goose (Block):** Extensibilidade total e execução de LLM.

## Protocolo de Assimilação
O LOGOS-X utiliza os `instructionsUrl` para realizar o "Mirroring" (espelhamento) de capacidades.
