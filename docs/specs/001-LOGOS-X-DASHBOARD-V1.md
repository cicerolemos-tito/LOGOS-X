## 1. Overview

### 1.1 Summary
A unified Web Application to visualize sales analytics (Big Bags, Slings, Laundry) and manage LOGOS-X squad operations. It transitions from static Excel data to a dynamic real-time dashboard.

### 1.2 Goals
- Visualize historical sales data (2018-2026).
- Display 2026 projections for Slings, Big Bags, and Laundry.
- Integrate LOGOS-X squad status and agent command center.
- Provide an API-first backend ready for future Flutter (V2) migration.

### 1.3 Non-Goals
- Native Mobile App (deferred to V2).
- Direct Excel editing within the web interface.
- Complex user authentication (use local simple auth for V1).

## 3. Technical Approach

### 3.1 Architecture Overview
- **Backend:** Node.js (Express) or Python (FastAPI) to parse Excel and serve JSON.
- **Frontend:** React + Tailwind CSS + Lucide Icons + Recharts for data visualization.
- **State Management:** Zustand (Standard LOGOS-X).

### 3.2 Key Decisions
- **Decision:** Use FastAPI (Python) for Backend.
- **Rationale:** Better performance and native libraries for Excel processing (pandas/openpyxl).

### 3.3 Patterns to Use
- **Repository Pattern:** To abstract Excel data access.
- **Clean Architecture:** To separate business logic (projections) from framework delivery.
